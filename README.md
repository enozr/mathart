案例都有代码实现(html+css+js)，web相关的技术社区庞大、受众广泛、标准清晰，也易于在pc、pad、手机等设备展示和传播。 不过，代码不是本文关注的核心，技术日新月异，原理大同小异，我希望为读者呈现一些有趣、有用的原理和简单的代码实现。

用数学和代码展现美，这里依旧是一片激动人心的荒原，come up！
